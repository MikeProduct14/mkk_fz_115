## Как тестировать проверки

1. Откройте любую ссылку выше
2. На карточке клиента нажмите кнопку **"Запустить проверку"**
3. Система автоматически:
   - Отправит запрос к NewDB API для проверки по перечню РФМ
   - Отправит запрос к NewDB API для проверки паспорта МВД
   - Сохранит результаты в таблицу `client_checks`
   - Обновит статус клиента на основе результатов
4. Результаты отобразятся на карточке клиента через 2-5 секунд

## Возможные результаты

- ✅ **Не найден** (clear) - клиент прошел проверку
- ❌ **НАЙДЕН** (found) - клиент в перечне РФМ или паспорт недействителен → статус "Отклонён"
- ⚠️ **Требует ручной проверки** (error) - API недоступен или ошибка → статус остается "Черновик"

## Проверка настроек API

Убедитесь что в `.env.local` указан ключ NewDB API:
```bash
NEWDB_API_KEY=ваш_ключ_здесь
```

Если ключ не указан, все проверки вернут ошибку 422.
