## 🚀 Как начать тестирование

### Шаг 1: Настройте Supabase (5 минут)

Следуйте инструкции в `SUPABASE_SETUP_GUIDE.md`:

1. Создайте проект на https://supabase.com
2. Выполните SQL миграцию из `supabase/migrations/001_initial_schema.sql`
3. Скопируйте URL и API ключ
4. Обновите `.env.local`:

```env
NEXT_PUBLIC_SUPABASE_URL=https://ваш-проект.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=ваш-ключ
```

### Шаг 2: Проверьте подключение

```bash
node test-supabase.js
```

Должно быть:
- ✅ Переменные окружения настроены
- ✅ Подключение к БД успешно
- ✅ Таблица organizations существует

### Шаг 3: Перезапустите dev сервер

```bash
# Dev сервер уже запущен, но после обновления .env.local нужно перезапустить
# Нажмите Ctrl+C в терминале, затем:
npm run dev
```

### Шаг 4: Откройте приложение

http://localhost:3000

### Шаг 5: Пройдите тестирование

Используйте `TESTING_CHECKLIST.md` для полного тестирования всех функций.
