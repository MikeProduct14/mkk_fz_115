# Статус проекта Compliance MFO

## ✅ Реализовано

### Инфраструктура
- [x] Next.js 14 с App Router и TypeScript
- [x] Supabase интеграция (клиент + сервер)
- [x] shadcn/ui + Tailwind CSS
- [x] Middleware для защиты роутов
- [x] Схема БД с RLS политиками

### Аутентификация
- [x] Страница регистрации (`/register`)
- [x] Страница входа (`/login`)
- [x] Защита dashboard роутов
- [x] Выход из системы

### Dashboard
- [x] Layout с sidebar навигацией
- [x] Главная страница с статистикой
- [x] Статус compliance (профиль МФО, ПВК)
- [x] Счётчики клиентов по статусам

### Страницы (заглушки)
- [x] Список клиентов (`/dashboard/clients`)
- [x] Новая анкета клиента (`/dashboard/clients/new`)
- [x] Карточка клиента (`/dashboard/clients/[id]`)
- [x] Страница ПВК (`/dashboard/pvk`)
- [x] Настройки организации (`/dashboard/settings`)

### API Routes (заглушки)
- [x] `/api/fns` - прокси ФНС ЕГРЮЛ
- [x] `/api/check` - проверки РФМ + паспорт МВД
- [x] `/api/pdf` - генерация PDF досье

### База данных
- [x] Таблица `organizations` - профили МФО
- [x] Таблица `clients` - досье заёмщиков
- [x] Таблица `client_checks` - история проверок
- [x] Таблица `client_history` - версионирование анкет
- [x] RLS политики для изоляции данных
- [x] Индексы для производительности

### Валидация
- [x] Zod схема для анкеты клиента
- [x] TypeScript типы для БД

```bash
# Настройка .env.local
cp .env.local.example .env.local
# Заполните переменные окружения

# Запуск
npm run dev
```
