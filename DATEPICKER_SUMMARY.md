# Исправление DatePicker - Краткая сводка

## Что было исправлено

### ✅ Проблема 1: Дата смещалась на день назад
**Причина:** Использование `toISOString().split('T')[0]` приводило к смещению из-за timezone

**Решение:** Все операции с датами теперь в UTC:
- Создание: `new Date(Date.UTC(year, month, day, 0, 0, 0, 0))`
- Форматирование: через `getUTCDate()`, `getUTCMonth()`, `getUTCFullYear()`
- Конвертация в строку: через утилиту `dateToString()` вместо `toISOString()`

### ✅ Проблема 2: Нельзя выбрать год через dropdown
**Решение:** Уже было исправлено добавлением `captionLayout="dropdown-buttons"` в Calendar
