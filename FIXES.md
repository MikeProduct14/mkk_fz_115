# Исправления формы клиента

## 1. DatePicker полностью переработан ✅

### Что было:
- Точки между днями, месяцами и годами не проставлялись автоматически
- Нельзя было выбрать год через dropdown
- Input был внутри PopoverTrigger (плохой UX)

### Что стало:
- ✅ Автоматическая вставка точек при вводе (15032024 → 15.03.2024)
- ✅ Dropdown для выбора года (1900 - текущий год)
- ✅ Отдельный Input с иконкой календаря справа
- ✅ Placeholder "дд.мм.гггг" для понятности
- ✅ Ограничение на 10 символов (maxLength)
- ✅ Валидация при потере фокуса (blur)
- ✅ Автоочистка невалидных дат

### Как работает:
1. Вводите цифры: `15032024`
2. Автоматически форматируется: `15.03.2024`
3. При достижении 10 символов дата парсится и сохраняется
4. Или кликните на иконку календаря справа для выбора через UI

## 2. Select исправлен (controlled/uncontrolled warning) ✅

### Проблема:
```
Base UI: A component is changing the uncontrolled value state of Select to be controlled.
```

### Причина:
Select получал `undefined` как начальное значение, затем менялся на строку.

### Решение:
```typescript
// Было:
value={watch('loan_purpose') ?? undefined}

// Стало:
value={watch('loan_purpose') || ''}
onValueChange={(value) => setValue('loan_purpose', value || undefined)}
```

Теперь Select всегда controlled (начинается с пустой строки '').
